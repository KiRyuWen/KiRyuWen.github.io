## Height (Confusing)
### Height of Tree (counts the node from root)
從目前Root(含root)的點+一路數到最深的leaf  
### Height of node (counts the edge which selected the node then go down lowest leaf along paths)
從你選的Node一路計算到最深的leaf的"edge"的數量
## Depth (current Node how far from root)
從目前Node 數回最上層root的edge數量
